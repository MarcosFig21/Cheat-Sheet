HTML Basics

Tags Name Description
< p > Paragraph Used to create a paragraph
< h > (h1-h6) Header Used to create a header where 1 is the largest and 6 is the smallest
< br > Break Used to break sentances in paragraphs
< !DOCTYPE html > Document declaration Necessary to have at the beginning of any HTML document
< html > HTML Used to define the whole HTML document
< head > Document head Needed to declare the head of the document
< title > Title Placed inside the document head inorder to give your webpage a title in the tab (not the actual page title)
< body > Body Needed to declare the body of the document
< footer > Footer Needed to decalre the footer of the document
< a > Anchor HTML links are define with the anchor tag
< img > Image HTML images are degine with the image tag
href href located inside the opening anchor tag the href is used to define the link
src src located inside the image or media tag, src is used to define the content being displ

HTML Tables

Tags Name Description
< table > Table Used to define an HTML table
< thead > Table Head Used to define the head of the table
< tbody > Table Body Used to define the body of the table
< tfoot > Table Footer Used to define the footer of the table
< tr > Table Row Used to define a row in the table
< td > Table Data Placed inbetween the table row and is used to define data added to the table
colspan Column Span Defines how many columns a table data should span
rowspan; Row Span Defines how many rows a table data should span

CSS Selectors and Specificity

Tags Name Description
body Body used to select every element in the body
h1-6 Heading Used to select every heading in the document from h1 to h6
#id-name ID Used to select specific sections in an HTML document where an ID has been assigned to it
.class-name Class Used to select specific sections in an HTML document where a Class has been assigned to it
* Universal Used to select all elements insde an HTML document
h1, p, h2 (examples) Grouping Used to select multiple elements in an HTML document that require the same styling using 1 line of CSS code
element(p).class-name Specific Selector Class Used to target specific elements in this example a paragrah that has a specific class assigned to it

CSS Properties

Tags Name Description
color: color Used to change the color of the text or foreground of an element
background-color: Background Color Used to change the background color of an element
font-weight: Font Weight used to set the weight or the thickness of the font for example making text bold
margin: Margin Used to determine the size of the margin for an element
margin-top/right/bottom/left: Margin Self explanitory
font-family: Font Family Used to determine the font family of a text element for exampl serif
font-size: Font Size Used to determine the size of the font
border: Border Used to give an element a border
font-style: Font Style Used to determine the style of the font used for example italic
border-width: Border Width Self explanitory
letter-spacing: letter Spacing Self explanitory
word-spacing: Word Spacing Self explanitory
max-width: Max Width Self explanitory
min-width: Minimum Width Self-explanitory
padding: Padding Used to determine the padding of an element
padding-top/right/bottom/left: Padding self-explanitory
opacity: Opacity used to determine the opacity in HTML elements
